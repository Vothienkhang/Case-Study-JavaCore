# Product Management System

This is a simple **Product Management** application written in **Java**.  
The project demonstrates basic concepts such as **OOP**, **file handling**, **exception handling**, and **menu user interaction**.

## Features
- Add, update, delete products  
- Display product list
- Search and Sort
- Save and load product data from file  
- Input validation and exception handling  

## Technologies
- Java Core
- File I/O (FileReader, FileWriter / BufferedReader)
